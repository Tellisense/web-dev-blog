Colors in CSS:

To use basic colors in css all you need is to type the name of the color
To use the color red, type:
color: red
To use the color green, type:
color: green
To use the color blue, type:
color: blue
Some more examples of color names:
color: yellow
color: purple
color: coral
You can also use the hexadecimal color value with #RRGGBB
using the numbers 0-9 and letters A-F
To use the color red, type:
color: #FF0000
To use the color green, type:
color: #00FF00
To use the color blue, type:
color: #0000FF
Some more examples of hexadecimal colors:
color: #5F9EA0
color: #DEB887
color: #2F4F4F
You can also use the RGB color value with rgb(rrr,ggg,bbb)
using the numbers 0-250
To use the color red, type:
color: rgb(255,0,0)
To use the color green, type:
color: rgb(0,255,0)
To use the color blue, type:
color: rgb(0,0,255)
Some more examples of using rgb colors:
color: rgb(255, 215, 0)
color: rgb(238, 130, 238)
color: rgb(154, 205, 50)
Now if you would like to change the opacity you can add any number from 0-1 in place of the a.
rgb(rrr,ggg,bbb,a)
To make a more translucent red, type:
color: rgba(255,0,0,.3)
To make a more solid red, type:
color: rgba(250,0,0,.7)
