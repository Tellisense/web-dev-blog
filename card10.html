topic

the bootstrap 12 grid system, uses, functions and applications.

overview

this system offers unparalleled simplicity in regard to positioning by dividing the UI into 12 horizontal columns (col).

syntax

the way this is done is by specifying how many of the 12 columns you want a given element to span. 3 for a fourth, 6 for half, 12 for full etc.

how to

place the following code within the class of a given elements "col- {the number of columns you want your element to span (1-12)}"

screen size compatibility

full utilization of the 12 grid system is also a great way arrange the elements on the web page in a way that is responsive to the users screen size by specifying how many of the 12 columns you want a given element to span in relation to screen sizes (xl, lg, md, sm sx). example: col-lg-3 col-md-3 col-sm-6 col-xs-12.
