What is Embed?

The HTML <embed> element embeds external content at the specified point in the document. This content is provided by an external application or other source of interactive content such as a browser plug-in.

< To display a picture, it is better to use the <img> tag.
embed type="image/jpg" src="pic_trulli.jpg" width="300" height="200">

<embed type="video/webm" src="video.mp4" width="400" height="300">

Upload the video to YouTube Take a note of the video id Define an <iframe> element in your web page Let the src attribute point to the video URL Use the width and height attributes to specify the dimension of the player Add any other parameters to the URL
Example <iframe width="420" height="315" src="https://www.youtube.com/embed/tgbNymZ7vqY"> </iframe >
